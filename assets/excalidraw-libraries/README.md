# Excalidraw Libraries

This directory previously contained Excalidraw icon libraries for Azure architecture diagrams.

## Migration Notice

Azure architecture diagrams have been migrated to **Draw.io** via the
[simonkurtz-MSFT MCP server](https://github.com/simonkurtz-MSFT/drawio-mcp-server),
which has 700+ built-in Azure icons and does not require external icon libraries.

The Excalidraw skill is now repositioned for **hand-drawn whiteboarding,
brainstorming, wireframes, and informal sketches** only.

See `.github/skills/drawio/SKILL.md` for architecture diagram guidance.
